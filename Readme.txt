Нажмите на exe.7z, затем на огромную кнопку "Download" в середине экрана, чтобы скачать архив с приложением.

Приложение представляет собой домашнюю библиотеку, хранящую данные об имеющихся книгах.
Библиотека позволяет хранить информацию о названии книги, годе её выпуска, количестве
страниц, об издателе а так же об авторах этих книг, кроме того каждый уникальный пользователь
библиотеки имеет возможность хранить индивидуальный список закладок.

Для работы с приложением на рабочей машине должна быть установлена СУБД. 
Предварительная настройка системы под использование приложения показано на примере 
СУБД Oracle Database Express Edition 11g.

1. Запустите SQL Developer. Подключитесь к вашему локальному серверу с помощью учетной записи SYSTEM.
2. Создайте пользователя JDBC (ПКМ на папке Other Users -> Create User).
3. Назначьте ему роли RESOURCE,  CONNECT и привилегии CREATE VIEW,  UNLIMITED TABLESPACE.
4. Подключитесь к вашему локальному серверу с помощью учетной записи JDBC 
(ПКМ на Oracle Connections -> New Database Connection).
5. Запустите скрипт library.sql. Он создаст все необходимые объекты и заполнит данные.

После этого, загрузите архив exe.7z, распакуйте содержимое в любое удобное место, к примеру, в корень
диска C и перейдите в неё. Далее текстовым редактором откройте файл config.properties, после db.url
для корректной работы, если используете Oracle SQL, оставьте jdbc:oracle:thin:@localhost:1521:xe. 
После db.username должно быть JDBC, а после db.password - пароль, указанный вами при создании пользователя
базы данных. Сохраните файл и закройте его.

Откройте командную строку (на Windows 10 - откройте меню "Пуск" и на английской раскладке наберите cmd).
По умолчанию вы будете находится в C:\Users\(Ваш юзернейм) . Перейдите в папку exe, распакованную
из архива: если вы поместили папку "exe" из архива в корень диска C - то наберите в командной 
строке "cd C:\exe" без кавычек. Для начала работы с приложением напишите "java -jar library.jar".

Логин и пароль для "нулевого" пользователя/администратора - "admin" и "password" соответственно.
Успешно зайдя в систему, наберите "help" без кавычек, чтобы посмотреть список доступных команд.
